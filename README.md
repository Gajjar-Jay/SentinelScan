# NetRecon: Automated Vulnerability Scanner

## Overview
NetRecon is a lightweight, automated cybersecurity reconnaissance tool designed to map external network exposure. It scans target IPs and domains to identify open ports, discover running services, and highlight potential security vulnerabilities.

## Features (In Development)
* **Port Scanning:** Identifies open digital doors on a target system.
* **Banner Grabbing:** Detects the specific software versions running on open ports.
* **Vulnerability Reporting:** Generates clear "Where, What, How" remediation reports.

## Tech Stack
* **Backend Engine:** Python 3 (Socket programming)
* **Frontend Dashboard:** HTML5, CSS3, JavaScript
